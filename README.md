🛒 Cashier CLI (Python)

A Command Line Point of Sale (POS) application built with Python using Object-Oriented Programming (OOP). This project is currently under active development and will continue to receive new features.

✨ Current Features

- User Registration
- User Login
- JSON Database
- Automatic Database Creation
- Input Validation
- Login Attempt Limitation
- Automatic Account Ban after 3 Failed Password Attempts
- Handle Missing or Corrupted Database Files

🚧 Planned Features

- Product Management
- Inventory System
- Shopping Cart
- Cashier Transaction
- Purchase History
- Sales Report
- Admin Dashboard
- Customer Balance
- Password Hashing
- Multi User Support

🛠 Technologies

- Python 3
- Object-Oriented Programming (OOP)
- JSON Database
- Exception Handling
